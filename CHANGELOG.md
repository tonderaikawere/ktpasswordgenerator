# Changelog

All notable changes to the Kawerify Tech Passwords utility will be documented in this file.

## [1.0.0] - 2026-07-19

### Added
- Core password generation algorithm using cryptographically secure `window.crypto.getRandomValues`.
- Pronounceable phonetic syllable generator (Easy to Speak).
- Customizable character exclusion lists and standard presets (PIN, Memorable, Maximum Strength).
- Saved passwords vault stored locally in browser `LocalStorage`.
- multi-format exports for TXT, CSV, and JSON formats.
- Real-time strength analyzer (entropy calculations and estimated crack times).
- Responsive, premium dark glassmorphism design with animated background glow blobs.
- Educational Hub section teaching password entropy and security best practices.
