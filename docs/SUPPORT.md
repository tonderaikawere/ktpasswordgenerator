# Support Guide

This document outlines how to get assistance with the Kawerify Tech Passwords utility.

---

## 1. Troubleshooting Steps

Before reaching out for support, please review these common local troubleshooting steps:

### 1. Saved Passwords Missing
- Saved passwords are kept in your browser's local cache (`LocalStorage`).
- If you cleared your browser cookies, local cache, or site data, these passwords will be permanently deleted.
- If you use private browsing (Incognito) mode, saved passwords will not persist after you close the session.

### 2. Copy to Clipboard Not Working
- Ensure you have granted the website permission to access the clipboard API in your browser settings.
- Ensure your browser version supports the Clipboard API (most modern versions of Chrome, Safari, Edge, and Firefox support it fully).

## 2. Help Channels

If you encounter technical issues or code defects, you can seek help through these channels:

- **GitHub Issues**: Open a bug report or feature request on the [GitHub Repository Issues Page](https://github.com/tonderaikawere/ktpasswordgenerator/issues). Please describe your OS, browser version, and steps to reproduce.
- **Attribution Website**: For general questions about Kawerify Tech, please visit our website at [kawerifytech.com](https://kawerifytech.com).

Please note that this is a free, public-utility application. Support is provided on a best-effort basis by the community and Kawerify Tech team.
